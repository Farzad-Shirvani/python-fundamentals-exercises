# Example 17: GCD of Two Numbers
 
Given two integers `a` and `b`, calculate their greatest common divisor (GCD).

**Input:**  
- Two integers `a` and `b` in one line.  
- Constraint: `1 ≤ a, b ≤ 100000`

**Output:**  
- Print `gcd(a, b)`.

Example 1:
```
Input: 36 27
```
```
Output: 9
```

Example 2:
```
Input: 45 12
```
```
Output: 3
```
