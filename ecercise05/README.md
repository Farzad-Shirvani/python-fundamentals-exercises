# Exercise 5: Sum of Digits

Write a program that takes a **two-digit integer** as input and prints the **sum of its digits**.

Input: A two-digit integer `x` (10 ≤ x ≤ 99).  
Output: The sum of the digits of the input number.

 Example 1:
 ```
Input: 54
```
```
Output: 9
```
Example 2:
```
Input: 27
```
```
Output: 9
```
